# Figma Financial Dashboard Reference

V2 방향으로 쓰기 위해 저장한 Figma 커뮤니티 디자인 레퍼런스입니다. 기존 `/report`는 보존하고, 새 화면은 `/report-v2`에서 이 톤을 기반으로 진행합니다.

## Saved Files

- `financial-dashboard-main.png`: 메인 레퍼런스. 사이드바, KPI 카드, 지표 카드, 통계 패널의 전체 톤.
- `ecards-dashboard-table.png`: 보조 레퍼런스. 더 단순한 테이블 중심 카드/필터/좌측 네비 톤.
- `design-tokens.json`: 색상, 그림자, radius, 버튼/카드/사이드바 패턴.
- `tokens.css`: 프로젝트 화면에 복사해 쓰기 쉬운 CSS 변수.
- `component-notes.md`: 실제 적용 시 판단 기준과 컴포넌트별 사용법.

## Design Direction

Primary 레퍼런스가 데일리 리포트에 더 잘 맞습니다. 이유는 운영 화면에 필요한 “요약, 핵심 지표, 테이블, 코멘트, 보조 분석” 구조가 이미 있고, 밝은 금융 대시보드 톤이라 자금운용 리포트의 신뢰감과도 맞습니다.

Secondary 레퍼런스는 테이블 밀도와 필터/목록 톤을 참고할 때 쓰는 보조 자료로 둡니다.
