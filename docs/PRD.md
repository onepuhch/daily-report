# PRD: Daily Market Report Automation

## 문제
현재 데일리 리포트는 인포맥스 Excel Add-in으로 데이터를 가져오고, 사람이 숫자와 시장 코멘트를 확인해 이미지 형태로 공유한다. 이 방식은 반복 작업이 많고, 과거 데이터와 코멘트가 체계적으로 축적되지 않아 장기 분석이나 AI 검색에 활용하기 어렵다.

## 목표
- 기존 인포맥스 Excel 파일을 자동 갱신한다.
- 리포트에 필요한 숫자를 자동 추출한다.
- 날짜별 시장 데이터와 코멘트를 DB에 누적한다.
- Notion풍의 심플한 웹 리포트를 자동 생성한다.
- 관리자 화면에서 코멘트를 수정하고 발행한다.
- 장기적으로 RAG 챗봇과 답변 기반 차트를 제공한다.

## 사용자
- 데일리 리포트를 작성, 검수, 배포하는 실무자
- 카카오뱅크 내부에서 시장 현황을 조회하는 사용자
- 향후 과거 시장 코멘트와 데이터 검색을 원하는 사용자

## MVP 범위
1. `MARKET DAILY.xlsm` 자동 갱신 프로세스 설계
2. `camera` 시트 및 주요 원천 시트의 추출 대상 정의
3. Supabase PostgreSQL 데이터 모델 설계
4. HTML 리포트 디자인 초안
5. 코멘트 초안/최종본 저장 구조
6. 관리자 화면 요구사항 정의

## MVP 제외
- 텔레그램 자동 수집
- 완전 자동 발행
- 고도화된 챗봇
- API 기반 전체 데이터 대체
- 실시간 시장 데이터 대시보드

## 성공 기준
- 하루 리포트 생성에 필요한 수작업 시간이 크게 줄어든다.
- 숫자 데이터와 최종 코멘트가 날짜별로 누적된다.
- 리포트는 HTML로 조회 가능하고, 카톡 공유용 이미지 또는 링크를 만들 수 있다.
- 사람이 코멘트를 검수하고 최종 발행할 수 있다.
